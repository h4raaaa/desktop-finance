# Десктопное приложение "Менеджер личных финансов"

Приложение для учета доходов и расходов, построения отчетов и планирования бюджета.

## Структура проекта
backend
docs
frontend

## Функциональность

-Учет доходов и расходов
-Построение графиков и отчетов
-Категории транзакций
-Планирование бюджета
